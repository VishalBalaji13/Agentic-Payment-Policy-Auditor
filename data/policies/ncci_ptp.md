# National Correct Coding Initiative (NCCI) Procedure-to-Procedure (PTP) Edits

Source: Centers for Medicare & Medicaid Services (CMS), National Correct Coding
Initiative Policy Manual for Medicare Services. Publicly available at cms.gov.

This excerpt is illustrative and lightly simplified for demonstration purposes.

## Policy

The NCCI Procedure-to-Procedure (PTP) edits identify pairs of HCPCS/CPT codes
that should not, under most circumstances, be reported together for the same
beneficiary on the same date of service by the same provider. Each PTP edit pair
consists of a Column One code and a Column Two code. When a provider reports both
codes of an edit pair, the Column One code is eligible for payment and the Column
Two code is denied unless a clinically appropriate NCCI PTP-associated modifier is
reported with the claim.

Each edit pair carries a Modifier Indicator that governs whether a modifier may be
used to bypass the edit:

- Modifier Indicator 0: No modifier is allowed to bypass the edit. The Column Two
  code is denied when reported with the Column One code for the same patient on the
  same date of service, regardless of any modifier submitted.

- Modifier Indicator 1: A clinically appropriate modifier is allowed to bypass the
  edit when documentation supports that the two procedures were distinct or
  independent (for example, performed at separate anatomic sites, in separate
  encounters, or as separate sessions). Supporting documentation must be present in
  the medical record.

- Modifier Indicator 9: The edit does not apply and is not active.

When the Column Two code is denied under a PTP edit, the provider may not separately
report it; the work is considered bundled into the Column One code.

## Active Edit Pairs

The following PTP edit pairs are active under this excerpt:

1. Column One 80053 (Comprehensive metabolic panel) / Column Two 80048 (Basic
   metabolic panel) — Modifier Indicator 0. The basic metabolic panel is a
   component of the comprehensive metabolic panel and is not separately payable.
   No modifier may bypass this edit.

2. Column One 93000 (Electrocardiogram, complete, with interpretation and report) /
   Column Two 93005 (Electrocardiogram, tracing only, without interpretation) —
   Modifier Indicator 1. A clinically appropriate modifier may bypass the edit when
   documentation supports a distinct service.

3. Column One 36415 (Collection of venous blood by venipuncture) / Column Two 36416
   (Collection of capillary blood specimen) — Modifier Indicator 1. A clinically
   appropriate modifier may bypass the edit when the collections are independent.
